# Bikcraft – Site institucional (projeto de estudo)

Projeto de um site institucional para uma loja fictícia de bicicletas, desenvolvido durante um curso de Front-end com foco em prática de HTML semântico, CSS responsivo e JavaScript básico.

O objetivo principal deste projeto é consolidar fundamentos de desenvolvimento web, estruturação de páginas, responsividade e organização de código.

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte de um curso de Front-end.

O processo de desenvolvimento foi feito da seguinte forma:

- primeiro implementei as soluções por conta própria;
- em seguida, utilizei o conteúdo do curso como apoio apenas para validação, correção de dúvidas e melhoria de abordagem.

O layout e a estrutura seguem as instruções propostas no curso, servindo como base para o desenvolvimento das habilidades técnicas.

---

## 🎯 Objetivo

Treinar e consolidar:

- HTML semântico
- organização de layout
- responsividade
- fundamentos de JavaScript para interações básicas

---

## 🛠️ Tecnologias utilizadas

- HTML5 (semântico)
- CSS3 (layout e responsividade)
- JavaScript (funcionalidades básicas)

---

## 👨‍💻 Meu papel no projeto

Todo o desenvolvimento foi realizado por mim, incluindo:

- estruturação semântica das páginas
- estilização responsiva
- integração de scripts em JavaScript
- implementação do layout

O design e o código foram desenvolvidos seguindo as instruções do curso, com execução própria e validação posterior.
